# Assumptions

## 実装上の仮定

- **daily_limit_count**: 10回/日（仮設定。SPEC.md §6.2参照、後日DBで変更可能）
- **日付判定基準**: JST（UTC+9）（SPEC.md §7参照）
- **通知先**: コマンド実行チャンネル（SPEC.md §5.1の成功時メッセージ例より推定）
- **ポイント操作**: 付与のみ、減算なし（SPEC.md §4.1参照）
- **v1の有効ポイント種**: 'mero' のみ（SPEC.md §3.1、§4.1参照）

## 設定変更可能性

上記値は実装時の仮定であり、SPEC.md §6.2の通り初期データとして設定される。将来的にはDB設定値により変更可能な設計とする。
